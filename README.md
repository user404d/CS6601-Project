## Installation

#### Requirements

- Java `>=7`
- Ant `>=1.10`

```bash
git clone https://github.com/user404d/CS6601-Project.git
cd CS6601-Project
ant archive
```

## Usage

```bash
# ./runmillionaire <Value of Alice> <Value of Bob>
./runmillionaire 123 23
```

Verify the output in `results/alice.out` and `results/bob.out`


## Millionaire Circuit

```plain
.input a 1 32
.input b 2 32
yao gteu a b
.output yao
```
